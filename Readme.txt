Doodle Jump — игра легендарная. Она появилась в 2009 году, когда сам жанр игр для мобильных устройств находился в зачаточном состоянии. 
Главным героем игры Doodle Jump является маленький зеленый пришелец,вечно прыгающий по платформам на своих маленьких лапках.
Основной целью игры является ваша помощь так называемому «Дудлеру» (англ. Doodle) добраться по узким платформам, 
насколько это возможно, выше и набрать при этом как можно более большое количество игровых очков. 
Управление игрой осуществляется с помощью мыши:
ЛКМ-прыжок влево;
ПКМ-прыжок вправо.
